# Chaos Runner (Unity Game)

## Description
Chaos Runner is a fast-paced Unity game where the player moves forward while avoiding obstacles and shooting enemies.

## Features
- Player movement and camera follow
- Shooting system with bullets
- Enemy AI that shoots back
- Damage system with health tracking
- Obstacles like spike traps

## Scripts
- Bullet.cs – Handles shooting and movement of bullets
- EnemyDamage.cs – Handles damage when enemy is hit
- EnemyShooter.cs – Enemy shooting behavior
- CameraShake.cs – Adds camera effects
- SpikeTrapDamage.cs – Damages player on collision

## Assets
- 3D models from Sketchfab
- File types used: FBX and OBJ
